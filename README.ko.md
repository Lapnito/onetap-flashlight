## OneTap이란?

**OneTap**은 길을 막지 않는 즉시 손전등입니다. 한 번 탭하면 300밀리초 이내에 불이 켜집니다 — 스플래시 화면 없음, 로딩 없음, 가입 없음. Android와 iPhone 모두 지원. OneTap은 손전등이 잘해야 하는 일에만 집중합니다: 빠르게 켜지고, 필요한 만큼 켜져 있고, 깔끔하게 꺼집니다. 모스 부호 SOS 신호, 의도적으로 분리된 스트로브 모드, 자동 꺼짐 타이머, 홈 화면 위젯, Android의 빠른 설정 타일까지 포함합니다.

원칙은 하나: **손전등은 손전등이어야 한다**. 광고 없음, 인터넷 연결 없음, 분석 없음, 새벽 3시의 "별점 부탁" 팝업 없음, 카메라 플래시 외에는 어떠한 권한도 요구하지 않습니다.

## 주요 기능

### 즉시 점등
- **탭부터 점등까지 300ms 미만**
- **카메라 LED** 최대 밝기
- **화면 라이트** 부드러운 흰색, 따뜻한 흰색, 빨간색

### 비상
- **모스 SOS 신호** — 국제 `· · · — — — · · ·` 패턴
- **스트로브 모드** — 1–20 Hz, 우발적 작동을 막기 위해 의도적인 길게 누름 뒤에 배치

### 편의
- **자동 꺼짐 타이머** — 2, 5, 10, 20분
- **홈 화면 위젯** — 앱을 열지 않고 점등
- **빠른 설정 타일**(Android) — 가장 빠른 접근

### 개인정보 보호
- **인터넷 권한 없음** — 앱은 네트워크에 접속할 수 없습니다
- **분석 없음, 추적 없음**
- **광고 없음, 인앱 결제 없음, 구독 없음**
- **고대비 인터페이스**로 한 손 사용 가능

## 사용 시나리오

| 상황 | OneTap이 하는 일 |
|------|-------------------|
| 정전 | 위젯 탭 — 300ms 이내 LED |

| 콘서트 | 의도적 활성화 스트로브 |
| 아이 야간등 | 저밝기 따뜻한 흰색 |

## 작동 방식

**왜 그렇게 빠른가요?**
스플래시도 애니메이션도 네트워크 호출도 텔레메트리도 없습니다. 앱은 손전등 UI에서 바로 시작됩니다.

**정말 오프라인인가요?**
네. OneTap은 인터넷 권한 자체를 요청하지 않습니다.

**SOS 패턴은 어떻게 생겼나요?**
국제 모스 SOS — 짧게 3번, 길게 3번, 짧게 3번 — 분명한 리듬으로.

**스트로브가 길게 누름 뒤에 있는 이유는?**
깜빡이는 빛은 광과민성 간질이 있는 분에게 위험할 수 있습니다. 활성화는 반드시 의도적이어야 합니다.

## 자주 묻는 질문

**정말 무료인가요?**
네. 광고도 인앱 결제도 구독도 없습니다.

**어떤 권한이 필요한가요?**
Android에서는 카메라 권한(플래시용)만. iOS에서는 시스템 손전등 API. 그 외에는 없습니다.

**데이터를 수집하나요?**
아니요. 인터넷 권한도, 분석도, 외부 SDK도 없습니다.

**타이머가 배터리를 보호하나요?**
네. LED는 의식하지 못해도 전력을 소모합니다. 자동 꺼짐이 잘못 켜둔 상태로 남는 것을 막아줍니다.

**야간등으로 적당한가요?**
적당합니다. 저밝기 빨간색이나 따뜻한 흰색 화면 라이트, 20분 타이머.

**실제로 얼마나 빠른가요?**
일반 하드웨어에서 실행부터 LED 점등까지 300ms 미만. 위젯과 타일은 더 빠릅니다.

**스트로브를 왜 분리했나요?**
광과민성 간질이 있는 분의 안전을 위해서입니다.

**태블릿에서도 작동하나요?**
네 — 플래시가 없는 태블릿에서는 화면 라이트 모드만 사용 가능합니다.

**SOS는 진짜 모스인가요?**
네. 국제 표준 — 짧게 3번, 길게 3번, 짧게 3번.
