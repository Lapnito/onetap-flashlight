## OneTap とは？

**OneTap** は、邪魔をしない瞬時の懐中電灯です。1タップで光が300ミリ秒以内に点灯します — スプラッシュ画面なし、読み込みなし、サインアップなし。Android と iPhone に対応。OneTap は懐中電灯が得意とすべきことに集中します：素早く点く、必要な時間だけ点いている、きれいに消える。さらに、モールス符号の SOS 信号、意図的に分離されたストロボモード、自動オフタイマー、ホーム画面ウィジェット、Android のクイック設定タイルを備えています。

原則はひとつ：**懐中電灯は懐中電灯である**。広告なし、ネット接続なし、計測なし、深夜3時の「評価してください」ポップアップなし、カメラのフラッシュ以外の権限要求もなし。

## 主な機能

### 瞬時の光
- **タップから点灯まで300ミリ秒未満**
- **カメラLED** で最大の明るさ
- **画面ライト** はソフトホワイト、ウォームホワイト、レッド

### 緊急時
- **モールス SOS 信号** — 国際的な `· · · — — — · · ·`
- **ストロボモード** — 1〜20 Hz、誤作動を防ぐため長押しで明示的に起動

### 便利
- **自動オフタイマー** — 2、5、10、20分
- **ホーム画面ウィジェット** — アプリを開かずに点灯
- **クイック設定タイル**（Android）— 最速のアクセス

### プライバシー
- **インターネット権限なし** — アプリはネットに接続できません
- **計測なし、トラッキングなし**
- **広告なし、課金なし、サブスクなし**
- **高コントラストUI** で片手操作にも対応

## 仕組み

**なぜそんなに速い？**
スプラッシュなし、アニメーションなし、ネット呼び出しなし、計測なし。アプリは懐中電灯のUIで直接起動します。

**本当にオフライン？**
はい。OneTap はネット権限自体を要求しません。

**SOS の形は？**
国際モールス SOS — 短3つ、長3つ、短3つ — の認識しやすいリズム。

**ストロボがなぜ長押しなの？**
点滅光は光感受性てんかんのある方にリスクがあります。起動は意図的でなければなりません。

## よくある質問

**本当に無料ですか？**
はい。広告も課金もサブスクもありません。

**必要な権限は？**
Android ではカメラ（フラッシュ用）のみ。iOS ではシステムの懐中電灯 API のみ。それ以外は一切なし。

**データを集めますか？**
集めません。ネット権限なし、計測なし、サードパーティ SDK なし。

**タイマーは電池の保護になりますか？**
なります。LED は気付かずに点いていても電力を消費します。自動オフが切り忘れを防ぎます。

**常夜灯に向いていますか？**
向いています。低輝度のレッドかウォームホワイトの画面ライト、20分タイマー。

**実際どれくらい速い？**
通常のハードで起動から点灯までが300ms未満。ウィジェットとタイルはさらに速い。

**ストロボが分離されている理由は？**
光感受性てんかんのある方の安全のためです。

**タブレットでも動きますか？**
動きます — フラッシュのないタブレットでは画面ライトのみ利用できます。

**SOS は本物のモールスですか？**
はい。国際標準パターン — 短3つ、長3つ、短3つ。

[github.com/Lapnito/onetap-flashlight/issues](https://github.com/Lapnito/onetap-flashlight/issues) に issue を、または tom@lapnito.cz へメール。
