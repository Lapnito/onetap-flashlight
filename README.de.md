## Was ist OneTap?

**OneTap** ist die Sofort-Taschenlampe, die einfach funktioniert. Ein Tipp und das Licht ist in unter 300 Millisekunden an — kein Splash-Screen, kein Laden, keine Anmeldung. Verfügbar für Android und iPhone. OneTap konzentriert sich auf das Wesentliche: schnell anschalten, so lange leuchten wie nötig, sauber abschalten. Mit dabei: ein Morse-SOS-Signal, ein bewusst entkoppelter Stroboskop-Modus, ein Auto-Off-Timer, ein Homescreen-Widget und unter Android eine Schnellzugriff-Kachel.

Ein einziges Prinzip: **eine Taschenlampe sollte eine Taschenlampe sein**. Keine Werbung, keine Internetverbindung, keine Analytik, keine Bewerten-Pop-ups um 3 Uhr morgens, keine Berechtigungen über das Kamerablitzlicht hinaus.

## Hauptfunktionen

### Sofort-Licht
- **Unter 300 ms vom Tipp zum Licht**
- **Kamera-LED** für maximale Helligkeit
- **Bildschirmlicht** in sanftem Weiß, Warmweiß oder Rot

### Notfall
- **SOS-Morse-Signal** — internationales Muster `· · · — — — · · ·`
- **Stroboskop-Modus** — 1–20 Hz, bewusst hinter einem Long-Press, damit es nie zufällig auslöst

### Komfort
- **Auto-Off-Timer** — 2, 5, 10 oder 20 Minuten
- **Homescreen-Widget** — Licht ohne App-Start
- **Schnellzugriff-Kachel** (Android) — schnellster Zugang über die Schnelleinstellungen

### Datenschutz
- **Keine Internet-Berechtigung** — die App kann nicht ins Netz
- **Keine Analytik, kein Tracking**
- **Keine Werbung, keine In-App-Käufe, kein Abo**
- **Hochkontrast-Oberfläche** für Einhandbedienung

## So funktioniert's

**Warum so schnell?**
Kein Splash, keine Animation, keine Netzwerkanfrage, keine Telemetrie. Die App startet direkt im Taschenlampen-UI.

**Wirklich offline?**
Ja. OneTap fragt die Netzwerkberechtigung gar nicht erst ab.

**Wie sieht das SOS-Muster aus?**
Internationaler Morse-SOS — drei kurz, drei lang, drei kurz — in einem klar erkennbaren Rhythmus.

**Warum ist Stroboskop hinter einem Long-Press?**
Flackerlichter können bei photosensibler Epilepsie ein Risiko sein. Die Aktivierung muss bewusst sein.

## FAQ

**Wirklich kostenlos?**
Ja. Keine Werbung, keine In-App-Käufe, keine Abos.

**Welche Berechtigungen?**
Auf Android nur Kamera (fürs Blitzlicht). Auf iOS die System-Taschenlampen-API. Sonst nichts.

**Werden Daten erhoben?**
Nein. Keine Internet-Berechtigung, keine Analytik, keine Drittanbieter-SDKs.

**Schont der Timer den Akku?**
Ja. Das LED zieht Strom, auch wenn man's nicht merkt — der Auto-Off sorgt dafür, dass es nie versehentlich angelassen wird.

**Als Nachtlicht geeignet?**
Ja. Bildschirmlicht in Rot oder Warmweiß bei niedriger Helligkeit, Auto-Off auf 20 Minuten.

**Wie schnell wirklich?**
Unter 300 ms vom Start bis LED an, auf typischer Mittelklasse-Hardware. Widget und Kachel sind noch schneller.

**Warum ist Stroboskop separat?**
Aus Sicherheit — Flackern kann bei photosensibler Epilepsie Anfälle auslösen.

**Auf Tablets nutzbar?**
Ja — Tablets ohne Blitz haben aber nur die Bildschirmlicht-Modi.

**Ist das SOS echtes Morse?**
Ja. Internationales Standardmuster — drei kurz, drei lang, drei kurz.
