## Що таке OneTap?

**OneTap** — це миттєвий ліхтарик, який не плутається під ногами. Один тап, і світло вмикається менше ніж за 300 мілісекунд — без сплеш-екрана, без завантаження, без реєстрації. Для Android та iPhone. OneTap робить те, що повинен робити ліхтарик: вмикається швидко, світить стільки, скільки потрібно, і чисто гасне. Додатково — Morse-сигнал SOS, окремо захищений стробоскоп, таймер автоматичного вимкнення, віджет на головний екран та плитка швидких налаштувань на Android.

Один принцип: **ліхтарик має бути ліхтариком**. Жодної реклами, жодного підключення до Інтернету, жодної аналітики, жодних спливаючих вікон "оцініть нас" о 3 ранку, жодних дозволів понад доступ до LED камери.

## Основні можливості

### Миттєве світло
- **Менше 300 мс від тапу до світла**
- **LED камери** для максимальної яскравості
- **Світло екрана** у м'якому білому, теплому білому або червоному

### Надзвичайні ситуації
- **Сигнал SOS у Morse** — міжнародний шаблон `· · · — — — · · ·`
- **Стробоскоп** — 1–20 Гц, навмисно за довгим натисненням, щоб ніколи не вмикався випадково

### Зручність
- **Таймер вимкнення** — 2, 5, 10 або 20 хвилин
- **Віджет на головному екрані** — світло без відкриття застосунку
- **Плитка швидких налаштувань** (Android) — найшвидший доступ

### Конфіденційність
- **Без дозволу на Інтернет** — застосунок не може підключитися до мережі
- **Без аналітики, без стеження**
- **Без реклами, без покупок, без підписки**
- **Високий контраст** для роботи однією рукою

## Як це працює

**Чому так швидко?**
Без сплеш-екрана, без анімацій, без мережевих викликів, без телеметрії. Застосунок запускається одразу в інтерфейсі ліхтарика.

**Дійсно офлайн?**
Так. OneTap навіть не запитує дозвіл на Інтернет.

**Який вигляд має SOS?**
Міжнародний SOS у Morse — три коротких, три довгих, три коротких — у впізнаваному ритмі.

**Чому стробоскоп за довгим натисненням?**
Мерехтливе світло може бути небезпечним для людей зі світлочутливою епілепсією. Активація має бути усвідомленою.

## Часті запитання

**Застосунок справді безкоштовний?**
Так. Без реклами, покупок та підписок.

**Які потрібні дозволи?**
На Android лише доступ до камери (для спалаху). На iOS системний API ліхтарика. Більше нічого.

**Чи збирає дані?**
Ні. Без дозволу на Інтернет, без аналітики, без сторонніх SDK.

**Чи захищає таймер батарею?**
Так. LED споживає енергію, навіть якщо ти про це не думаєш — автоматичне вимкнення гарантує, що ліхтарик ніколи не залишиться ввімкненим випадково.

**Підходить як нічник?**
Так. Червоне або тепле світло екрана з низькою яскравістю, таймер на 20 хвилин.

**Наскільки швидко насправді?**
Менше 300 мс від запуску до увімкнення LED на типовому залізі. Віджет і плитка ще швидші.

**Чому стробоскоп окремо?**
Заради безпеки людей зі світлочутливою епілепсією.

**Чи працює на планшетах?**
Так — планшети без спалаху матимуть лише режими світла екрана.

**SOS — справжній Morse?**
Так. Міжнародний стандарт — три коротких, три довгих, три коротких.
