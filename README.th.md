## OneTap คืออะไร?

**OneTap** คือไฟฉายทันใจที่ไม่กีดขวาง แตะครั้งเดียวก็ติดในไม่ถึง 300 มิลลิวินาที — ไม่มีหน้าโหลด ไม่มีการลงทะเบียน รองรับทั้ง Android และ iPhone OneTap โฟกัสกับสิ่งที่ไฟฉายควรทำได้ดี: ติดเร็ว ติดอยู่นานเท่าที่ต้องการ และดับสะอาด พร้อมสัญญาณ SOS แบบรหัสมอร์ส โหมดสโตรบที่ตั้งใจแยกออกมา ตั้งเวลาดับอัตโนมัติ วิดเจ็ตหน้าจอหลัก และไทล์ "ตั้งค่าด่วน" บน Android

หลักการเดียว: **ไฟฉายต้องเป็นไฟฉาย** ไม่มีโฆษณา ไม่มีการเชื่อมต่ออินเทอร์เน็ต ไม่มีการเก็บสถิติ ไม่มีป๊อปอัปขอเรตติ้งตอนตี 3 และไม่ขอสิทธิ์อะไรนอกจากแฟลชกล้อง

## คุณสมบัติหลัก

### ติดทันใจ
- **น้อยกว่า 300 ms ตั้งแต่แตะถึงไฟติด**
- **LED กล้อง** ให้ความสว่างสูงสุด
- **ไฟหน้าจอ** ขาวนวล ขาวอุ่น หรือแดง

### ฉุกเฉิน
- **สัญญาณ SOS รหัสมอร์ส** — รูปแบบสากล `· · · — — — · · ·`
- **โหมดสโตรบ** — 1–20 Hz ตั้งใจซ่อนหลังการกดค้างเพื่อไม่ให้ทำงานโดยบังเอิญ

### ความสะดวก
- **ตั้งเวลาดับอัตโนมัติ** — 2, 5, 10 หรือ 20 นาที
- **วิดเจ็ตหน้าจอหลัก** — เปิดไฟโดยไม่ต้องเปิดแอป
- **ไทล์ตั้งค่าด่วน** (Android) — เข้าถึงเร็วที่สุด

### ความเป็นส่วนตัว
- **ไม่มีสิทธิ์อินเทอร์เน็ต** — แอปไม่สามารถเชื่อมต่อเครือข่ายได้
- **ไม่มีการเก็บสถิติ ไม่มีการติดตาม**
- **ไม่มีโฆษณา ไม่มีการซื้อในแอป ไม่มีการสมัครสมาชิก**
- **อินเทอร์เฟซคอนทราสต์สูง** เหมาะกับการใช้มือเดียว

## ทำงานอย่างไร

**ทำไมเร็วขนาดนั้น?**
ไม่มีหน้าโหลด ไม่มีอนิเมชัน ไม่มีการเรียกเครือข่าย ไม่มีเทเลเมทรี แอปเปิดมาที่หน้าไฟฉายเลย

**ออฟไลน์จริง ๆ ใช่ไหม?**
ใช่ OneTap ไม่ขอสิทธิ์อินเทอร์เน็ตเลย

**รูปแบบ SOS เป็นอย่างไร?**
SOS รหัสมอร์สสากล — สั้น 3 ครั้ง ยาว 3 ครั้ง สั้น 3 ครั้ง ในจังหวะที่จำได้

**ทำไมสโตรบต้องกดค้าง?**
แสงกะพริบอาจเป็นอันตรายกับคนที่เป็นโรคลมชักไวต่อแสง การเปิดต้องเป็นไปอย่างจงใจ

## คำถามที่พบบ่อย

**แอปฟรีจริงไหม?**
ใช่ ไม่มีโฆษณา ไม่มีการซื้อในแอป ไม่มีการสมัครสมาชิก

**ต้องขอสิทธิ์อะไรบ้าง?**
Android ขอเฉพาะสิทธิ์กล้อง (เพื่อใช้แฟลช) ส่วน iOS ใช้ API ไฟฉายของระบบ ไม่มีอย่างอื่น

**เก็บข้อมูลไหม?**
ไม่ ไม่มีสิทธิ์อินเทอร์เน็ต ไม่มีสถิติ ไม่มี SDK ของบุคคลที่สาม

**ตั้งเวลาช่วยรักษาแบตหรือเปล่า?**
ช่วย LED กินไฟแม้คุณจะไม่ทันสังเกต การดับอัตโนมัติช่วยไม่ให้ลืมเปิดไว้

**ใช้เป็นไฟกลางคืนได้ไหม?**
ได้ ไฟหน้าจอแดงหรือขาวอุ่นความสว่างต่ำ ตั้งเวลา 20 นาที

**เร็วจริงแค่ไหน?**
ต่ำกว่า 300 ms ตั้งแต่เปิดแอปจนถึง LED ติดบนฮาร์ดแวร์ทั่วไป วิดเจ็ตและไทล์เร็วกว่านั้น

**ทำไมแยกสโตรบออก?**
เพื่อความปลอดภัยของคนที่เป็นโรคลมชักไวต่อแสง

**ใช้งานบนแท็บเล็ตได้ไหม?**
ได้ — แท็บเล็ตที่ไม่มีแฟลชจะใช้ได้แค่โหมดไฟหน้าจอ

**SOS เป็นมอร์สจริง?**
จริง รูปแบบมาตรฐานสากล — สั้น 3 ครั้ง ยาว 3 ครั้ง สั้น 3 ครั้ง

**แจ้งบั๊กอย่างไร?**
เปิด issue ที่ [github.com/Lapnito/onetap-flashlight/issues](https://github.com/Lapnito/onetap-flashlight/issues) หรืออีเมล tom@lapnito.cz
